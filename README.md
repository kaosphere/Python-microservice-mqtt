# Haptics project

Pour installer les dépendances : 

- Créer un virtual env : 

``` python3 -m venv venv ```

- Installer les dépendances : 

``` pip3 install -r requirements.txt ```


Pour lancer le projet : 

``` source venv/bin/activate ```

``` python3 main.py ```

** Attention, mosquitto doit être lancé au préalable **
